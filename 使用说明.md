# 使用说明

## 快速开始（推荐）

**只需一条命令：**
```cmd
setup_all.bat
```

这将自动完成所有工作！包括：
1. 初始化Git仓库
2. 编译程序
3. 运行测试
4. 生成提交包

## 生成的文件

运行后会生成：
- `MySolution.tar` - 这就是要提交的文件！

## 提交到平台

1. 找到 `MySolution.tar` 文件
2. 打开浏览器访问：http://10.176.56.208:5000
3. 上传 `MySolution.tar`

**重要：每天只能提交一次！**

## 主要文件

| 文件 | 说明 |

| MySolution.h | 头文件（会被打包） |
| MySolution.cpp | 实现文件（会被打包） |

## 算法说明

使用了 **HNSW** 算法（业界SOTA）：
- 多层图索引结构
- 快速的对数时间查询
- 高召回率

## 遇到问题？

编辑 `MySolution.cpp` 文件，在构造函数中修改
